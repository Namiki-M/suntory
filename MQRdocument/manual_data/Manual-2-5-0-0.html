1.6.1.1　SKU change work
a.　Preparation of SOP/check sheet
Defect mode Organoleptic defect / Mixing different liquid / Microbe contamination / Sealing defect / Packaging appearance defect
Subjected packaging All packaging
Subjected line All lines
Purpose
To correctly set the pipeline configuration, select the equipment parts, and input the setting values required for production to meet the designed production conditions before production
What/how to achieve

◆Create SOP for manual pipeline change, part change, and items that require manual equipment operation.

 　
　

・Prepare a check sheet to check and record work results so that all the required work is completed. (See the background.)

 　

◆It is desirable to install an interlock on manual pipeline change work and work that requires manual equipment operation.

 　

◆For CIP and production pipeline connection change, see the description about pipes in 1.3.6.2.

 　
How to maintain/control
Task 1
Procedure Use a check sheet to check and record that manual work and operation have been performed appropriately. 　
Frequency or timing Every work 　
Operating limit or standard condition The production conditions must be formed/established. 　
Background
Past incident: It is important to check using a check sheet and an interlock to prevent the following incidents:

1. Incident due to swing bend misconnection (two cases)


- CIP chemical mixture into the production line due to misconnection of a swing bend used to switch between the CIP pipeline and the production pipeline
- Carbonic acid mixture into non-carbonated products due to misconnection of a swing bend used to switch between carbonic acid gas and nitrogen gas
In both cases, misconnection was not noticed because there was no check sheet or interlock.

2. Incident due to a manual operation failure of equipment


In a line where both carbonated and non-carbonated products can be produced, either carbonic acid gas or nitrogen gas had to be selected for each production SKU.
In non-carbonated product production, carbonic acid gas was selected by mistake but never noticed because there was no check sheet.
