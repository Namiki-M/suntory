1.3.17.2　Inspection machines
a.　Equipment design
Defect mode Different packaging / Low or excessive filling / Lack of products in secondary packaging / Sealing defect / BBD coding defect / Foreign material contamination
Subjected packaging All packaging
Subjected line All lines
Purpose
To appropriately design equipment if an inspection machine is installed and assure that all the target products are to be inspected.
What/how to achieve

◆Products must be sufficiently inspected within the line speed range.

 　

◆To appropriately shoot the image of inspection targets, install a vibration prevention, select the camera and light source, and set the shooting angle of camera and illumination intensity. To secure the angle of view or remove defective products appropriately, cut pitches between products as necessary. (See the figure.)

 　

◆Prevent uninspected or defective products from outflowing to the downstream of the inspection machine.

 　
　

・Adopt a control to stop the conveyor when an inspection error or removal error happens.

 　
　

・Adopt a conveyor design so that removed products, which have been moved to the removal conveyor, are not bound back to the main process flow.

 　
　

・Adopt a control to trigger an alarm to notify before the rejection conveyor is full.

 　
　

・It is desirable to install a sensor to detect fallen products. (See the figure.)

b.　Operation management
Defect mode Different packaging / Low or excessive filling / Lack of products in secondary packaging / Sealing defect / BBD coding defect / Foreign material contamination
Subjected packaging All packaging
Subjected line All lines
Purpose
To appropriately manage equipment if an inspection machine is used for inspection and assure that all the target products are to be inspected.
What/how to achieve

◆A person in charge with technical knowledge must verify the equipment for each work (each product, each packaging, etc.) in advance and set the inspection standards.

 　

◆Any change in the inspection machine setting must be made along with a person in charge with technical knowledge.

 　

◆Standardize the procedures to identify and remove uninspected products remaining on the conveyor at an unplanned stop and restart the production.

◆In order to check the machine functions, standardize how to create and maintain defective samples and how often to renew them. See the background.

 　

◆Standardize the operation so that the inspection machine settings specified for each product or packaging can be selected correctly when SKU is changed.

 　
How to maintain/control
Task 1
Procedure Check the removal operation using defective samples. 　
Make sure that the number of defective samples is correct after checking the removal operation. (This is to prevent defective samples from outflowing to the market. See the background.) 　
Frequency or timing Set locally (Example: Before start, every two hours, at end) 　
Operating limit or standard condition Non-conforming products must be removed appropriately, and the number of defective samples must be correct. 　
Task 2
Procedure Perform maintenance on the inspection machine based on the frequency recommended by the manufacturer (checking the link between the sensor and the remover, light source illumination, and camera position, and replacing the light source lamp, etc.). 　
Frequency or timing Set locally (Example: Every maintenance day, every OH) 　
Operating limit or standard condition Set locally (according to the frequency recommended by the manufacturer)
