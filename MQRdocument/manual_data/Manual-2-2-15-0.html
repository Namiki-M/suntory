1.3.16.1　BBD coding
a.　General requirement
Defect mode BBD coding defect
Subjected packaging All packaging
Subjected line All lines
Purpose
To appropriately print all products.
For details on the requirements for IJP or laser printing, see the corresponding separate sheet.
What/how to achieve

◆Install a printer that can print best before dates satisfying the laws and regulations and print at least one best before date on all products. If there are no legal standards, each company must set the printing standards so that customers can read printed information correctly. (See the figure.)

 　

◆Take design measures to prevent print errors due to vibration around the printer.

 　

◆Verify in advance that printed information does not disappear and is appropriately applied (including the performance check of the water drop remover). (For details, see 1.3.16.b and 1.3.16.c.)

 　

◆Remove water drops from printing area of products before the printer.

 　

◆The print check operation must be standardized to ensure that there are no errors on the specified best before date print. (See the figure.) Clearly standardize the check, especially when the date is changed.

 　

◆Products with print errors must be removed by the inspection machine. Or print best before date two or more times, and ensure no print errors by sampling them.

 　

◆Predetermine the range of products to be removed when the printer makes an unplanned stop and standardize the removal operation.

 　

◆Fallen products must not pass through the printer. (Example: Removing on the curve conveyor. See the figure.)

 　

◆It is desirable to code filling time, filling valve No., capper head No., and etc. on products so that the production status can be traced. To reduce the difference between the filling time and the print time, it is desirable to code after the filler if possible.

 　
How to maintain/control
Task 1
Procedure Make sure that the actual print matches the planned print. 　
Frequency or timing Set locally (Example: When changing the date, before the start of production, before start after changing packagings) 　
Operating limit or standard condition All characters must add up perfectly. 　
Task 2
Procedure Check the printed information and the printing state on the products. 　
Frequency or timing Set locally (Example: Every two hours) 　
Operating limit or standard condition No incorrect date, character missing, or ink bleeding 　
Task 3
Procedure Start the water drop remover and check the spraying state. 　
Frequency or timing Set locally 　
Operating limit or standard condition Set locally

b.　Ink jet printer
Defect mode BBD coding defect
Subjected packaging All packaging
Subjected line All lines
Purpose
To appropriately design equipment and manage operation when printing with an ink jet printer
What/how to achieve

◆Use ink approved by the laws and regulations.

 　

◆The equipment must be stopped when ink is clogged or running out. Predetermine the range of products to be removed when it happens.

 　

◆Verify in advance that printed information does not disappear in the post-print process conditions (interference with the guide, contact with steam or cooling shower, etc.) when one of the following changes is made. See the background. The verification method must also be standardized.
　

・IJP setting change

 　
　

・Print location-related packaging change

 　
　

・Ink type change

 　
　

・Change in process conditions (guide width/height, steam, cooling water, etc.) that contact with the print location

 　
How to maintain/control
Task 1
Procedure Perform ink replenishment and cleaning. (To prevent ink from being clogged or running out) 　
Frequency or timing Set locally 　
Operating limit or standard condition Set locally 　
Task 2
Procedure If the equipment makes an unplanned stop due to ink clogging, securely remove products in the predetermined range from the process. 　
Frequency or timing When a print error occurs 　
Operating limit or standard condition No print errors. Unprinted products must not be present in the line after the printer. 　
Background
c.　Laser printing
Defect mode BBD coding defect / Sealing defect / Microbe contamination
Subjected packaging All packaging
Subjected line All lines
Purpose
To appropriately design equipment and manage operation when printing with a laser.
What/how to achieve

◆Verify in advance that printed information is readable and no pinholes are found on packaging when one of the following changes is made. See the background. The verification method must also be standardized.
　

・Change in the laser output or change in the distance between the light source and the print location

 　
　

・Packaging change. Pay extra attention when lighter packaging is used.

 　
　

・Blow molding condition change

 　

◆When the line is stopped, laser irradiation must be stopped (to prevent holes).

 　

◆It is desirable not to generate cross points by laser printing (multiple laser irradiation) if possible.

 　
How to maintain/control
Task 1
Procedure The laser output must be within the standard value range. 　
Frequency or timing Set locally (Example: Before start, every shift) 　
Operating limit or standard condition Set locally 　
Background
