1.3.9.1　Check before using packaging
a.　Check before using packaging
Defect mode Different packaging / Sealing defect / Difficult to open / Low or excessive filling / Others (violation of law) / Packaging appearance defect
Subjected packaging All packaging
Subjected line All lines
Purpose
To check packaging before use to prevent different or defective packaging from being used
To secure the traceability of used packaging
What/how to achieve

◆Make sure that the packaging is correct for the target SKU before use.

 　
　

・Be sure to perform this operation, especially for primary packaging and packaging on which the raw material and nutrient information is printed, because using different packaging is illegal.

 　
　

・Prepare a correct sample to check the information of printed packaging against it.

 　
　

・Specify a unique packaging name and code for each packaging and confirm correspondence between the planned packaging and the actual packaging. (Do not use the same name/code for any other packaging.)

 　
　

・It is desirable not to use similar packaging names because they may be mixed up even though packaging names/codes are unique. (In the past, packaging names "XXX1690" and "XXX1960" were mixed up.)

 　

◆If packaging needs to be placed near the manufacturing line, clearly separate the locations for the current production SKU and the next production SKU. If circumstances do not allow it due to space limitations, clearly indicate the difference to prevent misuse. (See the figure.)

 　

◆Make sure that there is no damage or strange odors on the packaging before use.

 　
How to maintain/control
Task 1
Procedure Make sure visually or using a system such as a barcode reader that the name and code of used packaging match the description in the production order sheet. (See the figure for how to check the information.) 　
Frequency or timing When setting the packaging in the equipment
Frequency: Roll label = Each roll, Can body = Each pallet, each pack		 　
Operating limit or standard condition The correct packaging must be used. 　
Task 2
Procedure For printed packaging, make sure that the printed information of prepared packaging matches that of the correct sample. (See the figure for how to confirm correspondence.) 　
Frequency or timing When setting the packaging in the equipment
Frequency: Be sure to check before production and when changing packaging SKU. It is desirable to check for each unit of packaging. (Example: Each roll of label)		 　
Operating limit or standard condition The correct packaging must be used. 　
Task 3
Procedure Record the items set locally about the use of packaging. 　
Frequency or timing Before setting the packaging in the equipment 　
Operating limit or standard condition All the items set locally must be recorded. (See the figure.)
