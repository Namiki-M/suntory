1.4.4.1　Physical/chemical test
a.　General requirement
Defect mode 　
Subjected packaging All packaging
Subjected line All lines
Purpose
To use and manage correct analysis equipment, instruments, and reagents and perform physical/chemical analysis in an appropriate environment.
What/how to achieve

◆Prepare a laboratory for physical/chemical analysis.

 　

◆Analysis must be performed in a correct order according to the manuals of analysis equipment, instruments, and reagents.

 　

◆Manage reagents appropriately while considering the following:
　

・Store each reagent in the corresponding storage method.

 　
　

・Perform inventory management based on the expiration dates.

 　
　

・For toxic or dangerous substances particularly, lock the storage area to prevent them from being lost or stolen.

 　
　

・Dispose of reagents in an appropriate method based on the laws and regulations.

 　
How to maintain/control
Task 1
Procedure Clean the laboratory and analysis equipment/instruments. 　
Frequency or timing Set locally 　
Operating limit or standard condition Set locally 　
Task 2
Procedure Perform inventory management for reagents. 　
Frequency or timing Set locally (Example: Once a week to check the stocks for any losses) 　
Operating limit or standard condition The actual stock number must match the description in the management list. 　
Task 3
Procedure Maintain and calibrate the analysis equipment. (See the figure.) 　
Frequency or timing Set locally (Example: See the frequency recommended by the manufacturer. In accordance with the laws and regulations. External calibration once a year) 　
Operating limit or standard condition Set locally
